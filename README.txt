* how to build
$ ant

** other targets
$ ant clean
$ ant rebuild

** requirements
- JDK 6 or later
- Ant (Eclipse-included also OK)


* how to execute
For executing, "twitter4j.properties" is required.
Template is "template.txt".

$ java -jar alice.jar [options...]


* tweet list
List is "list.txt".
Comments begin with "#".


* log
Logs will be output to "log/yyyymmdd.log".


* crontab
crontab example is "crontab.txt".
